A mobile application(similar to a famous Korean app called karrot), that is developed using react native. The app basically provides features like authentication, authorization, chat, item display, posting items to sell, posting content to the neighborhood, and commenting on posted content.

More Details about the featuresk
  • Authentication and Authorization
    • Custom authentication & Authorization -- Integrated a custom registration and sign in page, that allows users to authenticate and authorize themselves. JWT is used for users to access authorized contents of the App.
    • Verification and Reset account -- Integrated a sms verification method(using Twilio Service) during sign up and Nodemailer service is used to send reset codes when users forget their password when trying to sign in.
  • Chat Service features
    • Real Time message Exchange -- Integrated a chat feature where users of the app can chat in real time.I used Socket.IO for the communication to happen in real time.
    • Real Time functionalities -- Users can delete a messageare, and users will be displayed with typing... when the other person is typing.
  • General Interactions with the App
    • User infromation -- Users can update their profile, like items etc and real-time communication happens with the database, and user information is updated.
    • Item and content Post feature -- Users can post items they wish to sell and other contents or blogs. They can also comment on neighborhood posts, like the blog and every interaction is reflected in real time using Socket.IO.
