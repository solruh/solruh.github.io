Face Recognition Project

  • Tech Stack: Python, Pytorch, Google Colab, Servers with GPU

In this project the goal given multiple face images, detect. The project involves the following steps:

Steps or procedures of the project
  • Face Recognition Data Set collection
    • Collecting Baseline data -- Given list of korean names, Carried Out intensive data collection using python script that crawls through the internet and collect images for the given names. And then face detection for the collected face images(against standard image for each name) is run and all images with 0 or 2 detections are discarded.
    • Remove invalid images -- Embedding centroids for each person are extracted and compare all images to the centroid and if any image is quite far from the centroid(human check is made if image is valid or not) and multiple identities with the same name and duplicate identities are removed.
  • Experimenting with the base Model
    • CNN model and Loss Function -- Experimented with various loss functions like softmax loss, center loss and triplet loss aiming to decrease the distance between Anchor image and postive image and increase distance between anchor image and negative image. Also experimented with various CNN models like EfficientNet, ResNet18 aiming to achieve a higher accuracy.
    • Data augmentaton & Hyper parameter experiments -- Aiming to achieve lower EER, I did experiment with data augmentation and hyper paramter tweaking. The hyper parameters I tried to change in this project are learning rate, batch size, max epochs, optimizer, and scheduler's learning rate decay.In addition I tried several data augmentation methods such as color jitter(to play with the contrast, saturation and contrast of an image), random horizontal and vertical flip, gaussian blurring and random rotation.
  • Transfer Learning
    • Use open source general face data -- Did train the base model with vggface data set and saved the model weights to be used as an initial model weights in the Korean train data set.
