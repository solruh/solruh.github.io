E-commerce web app(mimics minal features of a famous korean E-commerce app called Coupang), that is developed using nextjs as a framework for frontend, nodejs for backend PostgreSQL as a database, and prisma as ORM. It has both Client View and Admin View. The app provides features listed below.

More Details about the featuresk
  • User Authentication and authorization
    • Next-Auth or Custom authentication -- Users will be able to authenticate themselves by registering or signing in using custom registration or next-auth, which provides easier authentication and authorization services for users to sign up or sign in using google or other service providers.
  • Client View Features
    • STT,Category & SubCategory, and ordinary search feature -- Integrated a feature where users can search the items they are looking for using various methods like typing in the search field, speaking to the mic, or hover around the allCategory Icon and they will be presented list of categories and sub-categories for each category and search accordingly.
    • Items Exploring feature -- Users are allowed to view item details, add Items to a cart, checkout the items in the cart, and leaving reviews along with starts for items at hand.
    • Payment Method -- Integrated a payment system(toss payments) that has large set of payment methods for seamless user experience during checking out.
  • Admin View Features
    • Dashboard -- Integrated an admin Dashboard, which helps admins to analyze their sells. Total Revenue made, total orders received, most sell percentage per category features are provided in graphical depiction using ChartJs and react-charts.
    • Products Feature -- Integrated a feature where admins can add new products to sell, Edit or Delte existing products.
    • Orders Feature -- Included an Orders tab, where admins can manage orders made to the items they posted. In this tab, an admin can change the status of the order(pending,shipped etc.) and delte or cancel the order made.
